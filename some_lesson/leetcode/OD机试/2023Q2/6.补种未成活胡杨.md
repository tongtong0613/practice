**题目**
```
近些年来，我国防沙治沙取得显著成果。某沙漠新种植N棵胡杨（编号1-N），排成一排。
一个月后，有M棵胡杨未能成活。
现可补种胡杨K棵，请问如何补种（只能补种，不能新种），可以得到最多的连续胡杨树？
```
**输入描述**
```
N 总种植数量
M 未成活胡杨数量
M 个空格分隔的数，按编号从小到大排列
K 最多可以补种的数量
其中：
1 <= N <= 100000
1 <= M <= N
0 <= K <= M
```
**输出描述**
```
最多的连续胡杨棵树
```
**示例1**
```
输入
5
2
2 4
1
输出
3
说明
补种到2或4结果一样，最多的连续胡杨棵树都是3。
```
**示例2**
```
输入
10
3
2 4 7
1
输出
6
说明
补种第7棵树，最多的连续胡杨棵树为6(5,6,7,8,9,10)
```
**代码**
```python
class Solution:
    def solve(self, n, m, no_trees, k):
        i = 1
        j = 1
        cur = 0
        max_len = 0
        while j <= n and i <= n and i <= j:
            while j <= n and (j not in no_trees or (j in no_trees and cur < k)):
                if j in no_trees:
                    cur += 1
                j += 1
            max_len = max(max_len, j-i)
            while i not in no_trees:
                i += 1
            cur -= 1
            i += 1
        return max_len


s = Solution()
print(s.solve(10,3,[2,4,10],2))

```
