**题目**
```
小朋友出操，按学号从小到大排成一列;
小明来迟了，请你给小明出个主意，让他尽快找到他应该排的位置
算法复杂度 要求不高于nlog(n); 学号为整数类型，队列规模 ≤ 10000;
```
**输入描述**
```
第一行: 输入已排成队列的小朋友的学号 (正整数) ，以”"隔开. 第二行: 小明学号
```
**输出描述**
```
输出一个数字，代表队列位置 (从1开始)
```
**示例1：**
```
输入
93,95,97,100,102,123,155
110
输出
6
```
**代码**
```python
class Solution:

    def solve(self, queue, id_no):
        queue_list = list(map(int, queue.split(',')))
        queue_list.append(float('inf'))
        n = len(queue_list)
        l, r = 0, n-1
        while l < r:
            mid = (l + r) // 2
            if queue_list[mid] > id_no:
                r = mid
            else:
                l = mid + 1
        return l + 1

s = Solution()
print(s.solve('93,95,97,100,102,123,155', 88))
```
