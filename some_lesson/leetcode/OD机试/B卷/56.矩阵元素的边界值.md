**题目**
```
给定一个N*M矩阵，请先找出M个该矩阵中每列元素的最大值，然后输出这M个值中的最小值
补充说明:
N和M的取值范围均为: [0,100]
```
**示例1：**
```
输入:
[[1,2],[3,4]]
输出:
3
说明: 第一列元素为: 1和3，最大值为3第二列元素为: 2和4，最大值为4各列最大值3和4的最小值为3
```
**代码**
```python
class Solution:
    def solve(self, input_str):
        stack = [float('-inf')] * 100
        n = len(input_str)
        i = 1
        cur = 0
        lenth = 0
        while i < n-1:
            if input_str[i] == '[':
                i += 1
            elif input_str[i] == ']':
                lenth = max(lenth, cur-1)
                cur = 0
                i += 2
            elif input_str[i] == ',':
                i += 1
            else:
                total = 0
                while input_str[i].isdigit():
                    total = total * 10 + int(input_str[i])
                    i += 1
                stack[cur] = max(stack[cur], total)
                cur += 1
        return min(stack[:lenth])


s = Solution()
print(s.solve('[[11,12],[13,14]]'))
```
