**题目**
```
给定两个字符串str1和str2，
str1进行 排列组合 只要有一个为str2的子串则认为str1是str2的关联子串，
请返回子串在str2的起始位置，若不是关联子串则返回-1。
```
**示例1**
```
输入
abc efghicbaiii
输出
5
```
**示例2** 
```
abc efghiccaiii
输出
-1
```
**代码**
```python
class Solution:
    def solve(self, str1, str2):
        import copy
        stack = list(str1)
        rec = -1
        i = 0
        n = len(str2)
        while i < n:
            tmp = copy.deepcopy(stack)
            if str2[i] in tmp:
                if rec == -1:
                    rec = i
                while tmp and str2[i] in tmp:
                    tmp.remove(str2[i])
                    i += 1
                if not tmp:
                    return rec
                else:
                    rec = -1
            else:
                i += 1
        return -1


s = Solution()
print(s.solve('abc', 'bdasdcbbfgs'))
```
