**题目**
```
给定一个整型数组，请从该数组中选择3个元素 组成最小数 字并输出（如果数组长度小于3，则选择数组中所有元素来组成最小数字）。
输入描述：
一行用半角逗号分割的 字符串 记录的整型数组，0 < 数组长度 <= 100，0 < 整数的取值范围 <= 10000。
输出描述：
由3个元素组成的最小数字，如果 数组长度 小于3，则选择数组中所有元素来组成最小数字。
```
**示例1：**
```
输入
21,30,62,5,31
输出
21305
说明
数组长度超过3，需要选3个元素组成最小数字，21305由21,30,5三个元素组成的数字，为所有组合中最小的数字
```
**示例2：**
```
输入
5,21
输出
215
说明
数组长度小于3，选择所有元素组成最小值，215为最小值
```
**代码**
```python
class Solution:
    def comp(self, a, b):
        if str(a) + str(b) < str(b) + str(a):
            return -1
        else:
            return 1

    def solve(self, input_str):
        nums = list(map(int, input_str.split(',')))
        nums.sort()
        nums = nums[:3]
        import functools
        nums = sorted(nums, key=functools.cmp_to_key(self.comp))
        output = ''
        for i in range(len(nums)):
            output += str(nums[i])
        return int(output)

s = Solution()
print(s.solve('21,30,62,5,31'))
```
