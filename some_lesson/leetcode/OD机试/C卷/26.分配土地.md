**题目**
```
从前有个村庄，村民们喜欢在各种田地上插上小旗子，旗子上标识了各种 不同的 数字
某天集体村民决定将覆盖相同数字的最小矩阵形的土地分配给村里做出巨大贡献的村民，请问此次分配土地，做出贡献的村民种最大会分配多大面
```
**输入描述**
```
第一行输入 m和n,
m 代表村子的土地的长
n 代表土地的宽
第二行开始输入地图上的具体标识
```
**输出描述**
```
此次分配土地，做出贡献的村民种最大会分配多大面积
备注
旗子上的数字为1~500，土地边长不超过500
未插旗子的土地用0标识
```
**示例1：**
```
输入
3 3
1 0 1
0 0 0
0 1 0
输出
9
说明
土地上的旗子为1，其坐标分别为(0,0)，(2,1)以及(0,2)，为了覆盖所有旗子，矩阵需要覆盖的横坐标为0和2，纵坐标为0和2，所以面积为9，即 (2
(2-0+1) = 9
```
**示例2：**
```
输入
3 3
1 0 2
0 0 0
0 3 4
输出
1
说明
由于不存在成对的小旗子，故而返回1，即一块土地的面积。
```
**思路**
```
1：题目很简单，先要理解【覆盖相同数字的最小矩阵形】，那么求这个值的思路就是找到所有的相同数字，然后找到每种数组的最大最小行列位置
就可以找到最小矩形的面积了。
2：然后就是从上述的所有最小矩形面积中找到 最大值 即可。
```
**代码**
```python
class Solution:

    def solve(self, m, n, matrix):
        min_row = [502] * 501
        min_col = [502] * 501
        max_row = [0] * 501
        max_col = [0] * 501
        exist = [0] * 501

        for i in range(m):
            for j in range(n):
                if matrix[i][j] != 0:
                    c = matrix[i][j]
                    exist[c] = 1
                    min_row[c] = min(min_row[c], i)
                    min_col[c] = min(min_col[c], j)
                    max_row[c] = max(max_row[c], i)
                    max_col[c] = max(max_col[c], j)
        max_value = 0
        for i in range(1, 501):
            if exist[i]:
                max_value = max(max_value, (max_col[i] - min_col[i] + 1) * (max_row[i] - min_row[i] + 1))
        return max_value


s = Solution()
print(s.solve(3, 3, [[1, 0, 1], [0, 0, 0], [0, 1, 0]]))
```
