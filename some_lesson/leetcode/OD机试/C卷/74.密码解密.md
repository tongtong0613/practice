**题目**
```
给定一段“密文”字符串 s，其中字符都是经过“密码本”映射的，现需要将“密文”解密并输出.
映射的规则 (a~i) 分别用 (1~ 9) 表示; (j~z) 分别用 (10*~26*) 表示
约束: 映射始终唯一
```
**输入描述**
```
“密文”字符串
```
**输出描述**
```
明文字符串
备注: 翻译后的文本长度在100以内
```
**示例1：**
```
输入
20*19*20*
输出
tst
```
**代码**
```python
class Solution:

    def solve(self, input_str):
        tmp = []
        n = len(input_str)
        i = n-1
        while i >= 0:
            if input_str[i] != '*':
                tmp.append(input_str[i])
                i -= 1
            else:
                tmp.append(input_str[i-2:i])
                i = i - 3
        ans = []
        for x in tmp:
            ans.append(chr(ord('a')+int(x)-1))
        return ''.join(ans[::-1])

s = Solution()
print(s.solve('110*22*222*1932'))
```
